# explore-tibet

Plan your Tibet pilgrimage — Lhasa's Potala Palace, Jokhang Temple, Namtso Lake, Everest Base Camp, and Tibetan Buddhist monastery visits.

## Overview

Plan your Tibet pilgrimage — Lhasa's Potala Palace, Jokhang Temple, Namtso Lake, Everest Base Camp, and Tibetan Buddhist monastery visits. Wraps the flyai-cli to provide real-time travel data with booking links.

## Quick Start

### Install Skill

```bash
clawhub install explore-tibet
# or
npx skills add alibaba-flyai/flyai-skill/tree/main/skills/explore-tibet
```

### Install CLI

```bash
npm i -g @fly-ai/flyai-cli
```

### Verify

```bash
flyai --version
```

## What It Does

Plan your Tibet pilgrimage — Lhasa's Potala Palace, Jokhang Temple, Namtso Lake, Everest Base Camp, and Tibetan Buddhist monastery visits.
